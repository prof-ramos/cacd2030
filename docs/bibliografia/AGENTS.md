Este arquivo serve como guia para agentes de IA trabalhando com as referências bibliográficas do edital do CACD 2030.

## Diretrizes para Agentes de IA

### Estrutura Organizacional
- **Manter organizado por matéria**: Cada referência deve ser classificada conforme as matérias do edital
- **Seguir estrutura do edital**: Seguir a hierarquia e formato das disciplinas oficiais
- **Níveis de prioridade**: Classificar conforme a importância para as fases do concurso

### Formato de Entradas
```markdown
## [Nome da Matéria]
- **Fonte:** [Tipo de material - Autor, Editora, Ano]
- **Fase:** [Fase do edital]
- **Prioridade:** [Crítica/Alta/Média/Baixa]
- **Descrição:** [Breve resumo do conteúdo]
```

### Critérios de Classificação
- **Fonte Oficial:** Material disponibilizado pelo Ministério das Relações Exteriores
- **Complementares:** Livros, artigos e documentos recomendados
- **Atualizadas:** Publicações recentes (últimos 5 anos)
- **Fundamentais:** Textos clássicos e indispensáveis para cada área

### Formato de Atualização
Sempre que adicionar novas referências:
1. Classificar pela matéria correspondente
2. Incluir informações completas de referência
3. Marcar prioridade conforme fase do concurso
4. Atualizar o arquivo bibliografia.md correspondente
