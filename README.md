## Fases do Concurso

1. **Primeira Fase** - Provas Objetivas
2. **Segunda Fase** - Provas Escritas
3. **Terceira Fase** - Exame Médico
4. **Quarta Fase** - Avaliação Psicológica
5. **Quinta Fase** - Prova Oral
6. **Sexta Fase** - Curso de Formação
